# Modulo-3-Reconocimiento

En la actividad se aprendió sobre reconocimiento en ciberseguridad utilizando herramientas como Kali Linux y Maltego.
Primero se realizó reconocimiento pasivo, utilizando Google Dorks para encontrar información pública en internet sin interactuar directamente con los sistemas. 
También se revisaron ejemplos en la base de datos de Exploit Database para identificar posibles exposiciones de información.
Después se utilizó Maltego para crear un grafo visual, agregando un dominio y obteniendo sus registros DNS, dirección IP y ubicación,
con el objetivo de visualizar cómo se conecta la infraestructura de un sitio web.
Finalmente se analizó la diferencia entre reconocimiento pasivo y activo, y se explicó que realizar estas actividades sin autorización puede ser ilegal, 
ya que podría considerarse acceso no autorizado o intento de ataque a sistemas informáticos.
